# 📚 RAG Book Tutor

An LLM-powered app to query PDF books using Retrieval-Augmented Generation (RAG).

## Features
- Long Summary
- Question Generation
- MCQ Generator

## Tech Stack
- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq (Llama 3.1)
- Streamlit

## Run Locally
pip install -r requirements.txt
streamlit run app.py
